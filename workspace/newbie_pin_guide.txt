This document provides all of the pin assignments

- Between Z430-2500T board and STM3204fxx.
z430 pin ---> STM3204fxx pin
P1  ---> GND
P2  ---> 3V
P13 ---> PB12
P15 ---> PB14
P16 ---> PB13
P17 ---> PB11
P18 ---> PB15

- Between Motor and Main Board (NOT WORKING)
Red Wire -> 3V
Black Wire -> GND
Yellow Wire -> PA10

- Between Proximity Sensor and Main Board
Red Wire -> 3V
Black Wire -> GND
Yellow Wire -> PA1
